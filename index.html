Prework Study Guide

HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The header element is at the top and contains dropdown menus, links etc.
  • The h1 element is the first element you use for your title.
  • The img element, along with src and alt, is used for inserting images.
  • The main element is where the bulk of the webpage is made.
  • Section elements are used to break up the page and make it easier to navigate.
  • The ul, ol and li elements are used for creating lists.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

Git

  • git status: checks what branch we are currently on.
  • git checkout -b branch-name: creates a new branch and switches to it.
